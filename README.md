# Лабораторна робота №3

### Можливі ключі
    --help, -h => довідка
    --version, -v => виводить версію

### Приклади
---
    $ ./run --help
    Usage:
      run <command> [options]
    Commands:
      version     Show version.
      help      Show help for commands.
---
    $ ./output --version
    run 1.0.1
---
